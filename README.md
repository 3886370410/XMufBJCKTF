# 项目简介 python576

本项目是一款基于 Python 的协同过滤商品推荐系统。

## 技术栈

- Python
- Django
- MySQL

## 系统角色

- 管理员
- 用户

## 功能模块

- 商品推荐
- 购物车
- 订单管理
- 用户评价
- 商品信息上传

## 运行环境

- Python 3.*
- Django 2.*
- MySQL 5.*

## 部署步骤

1. 安装 Python 和 Django
2. 配置 MySQL 数据库
3. 克隆或下载项目代码
4. 执行数据库迁移
5. 启动项目
